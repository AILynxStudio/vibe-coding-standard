# 图表生成说明

本目录包含 Vibe Coding 体系 **10 张可视化图表** 的详细绘图提示词。

## 使用方式

1. 打开任意 AI 绘图工具（如 DALL·E、Midjourney、Napkin AI、文心一格等）。
2. 复制对应 `.md` 文件中的完整提示词。
3. 粘贴发送，AI 会生成对应的示意图。
4. 如需统一风格，推荐在提示词前追加 `扁平化商务风格，白色背景，中文标签，配色统一`。

## 推荐生成参数

- 图片格式：PNG，透明或白色背景
- 分辨率：至少 1200×900px
- 风格：扁平化、简洁、商务，无照片写实元素

## 风格统一建议

为保证 10 张图视觉一致，所有提示词已内置以下统一参数：
- 画布：1200×900px（4:3）
- 主色：深蓝 `#1a365d`，辅色：中蓝 `#2b6cb0`、浅蓝 `#4299e1`、灰蓝 `#a0aec0`
- 字体：中文黑体或思源黑体，英文 Arial
- 线条：圆角矩形，箭头灰色 2px
