# 🚀 快速开始

## 一行命令安装

### Windows (PowerShell)
```powershell
git clone https://gitee.com/AILynx/vibe-coding-standard.git; cd vibe-coding-standard; .\quick-install.ps1
```

### Mac/Linux
```bash
git clone https://gitee.com/AILynx/vibe-coding-standard.git && cd vibe-coding-standard && chmod +x quick-install.sh && ./quick-install.sh
```

## 安装完成后

1. 打开 **Cursor** 或 **VSCode**
2. 创建一个新文件夹作为项目目录
3. 在AI对话框中输入 `/vibe-coding`
4. 按AI引导回答问题
5. AI会带你完成整个项目开发

## 就这么简单！

```
克隆 → 运行安装脚本 → 输入 /vibe-coding → 跟着AI走
```

## 常见问题

**Q: 安装后没反应？**
A: 重启Cursor或VSCode，然后重试

**Q: 找不到 /vibe-coding 命令？**
A: 确保skill已安装到正确位置：
- Cursor: `~/.cursor/skills/vibe-coding`
- VSCode: `~/.vscode/skills/vibe-coding`

**Q: 支持哪些AI工具？**
A: Cursor、VSCode + AI插件、桌面AI客户端、网页版AI
