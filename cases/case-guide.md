## 推荐学习路径

- **完全零基础**：案例0 → 案例1 → 案例2 → 案例3
- **有一点电脑经验**：案例1 → 案例2 → 案例3 → 案例4
- **想体验自动化**：案例5 → 案例2 → 案例3
- **想深入指挥AI**：案例3 → 案例4 → 案例6 → 案例7
