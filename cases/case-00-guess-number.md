# 案例0：5 分钟入门——用网页 AI 生成你的第一个网页游戏

> **难度**：☆（零基础，不需要安装任何东西）  
> **你需要**：一个浏览器，能访问 chat.openai.com 或 claude.ai  
> **你将做出**：一个在浏览器里能玩的"猜数字"小游戏

## 第 1 步：打开两个网页

- 左边：打开 **chat.openai.com** 或 **claude.ai**
- 右边：打开 **jsfiddle.net**（免费在线代码运行网站，不用注册）

## 第 2 步：把下面这段话复制给 AI

> 请帮我写一个简单的"猜数字"网页游戏。要求：
> 1. 电脑随机生成 1 到 100 之间的一个数字
> 2. 用户在输入框里猜，点"猜"按钮
> 3. 如果猜大了显示"太大了，再试试"，猜小了显示"太小了，再试试"
> 4. 猜对了显示"恭喜！你用了 X 次猜对了！"
> 5. 把全部 HTML、CSS、JS 写在一个 HTML 文件里，保证可以直接运行

（如果 AI 追问你用什么样式，就说"简单干净，白底深色字，按钮蓝色"）

## 第 3 步：AI 会给你一段代码

它看起来大概是这样（你不必看懂）：

```html
<!DOCTYPE html>
<html>
<head>
    <title>猜数字</title>
    <style>...</style>
</head>
<body>
    <h1>猜数字游戏</h1>
    <input type="number" id="guessInput">
    <button onclick="checkGuess()">猜！</button>
    <p id="message"></p>
    <script>...</script>
</body>
</html>
```



全选复制这段代码。

## 第 4 步：粘贴到 JSFiddle 并运行

1. 在 [jsfiddle.net](https://jsfiddle.net/) 页面，找到左上角的 **HTML** 框（不用管 CSS 和 JS 框）
2. 把 AI 给的代码完整粘贴进去
3. 点击页面左上角的 **Run** 按钮
4. 右下角 Result 区域就会出现一个猜数字游戏

## 第 5 步：趁热打铁，试着修改它

回到 AI 对话，继续告诉它你想怎么改（每次只提一个要求）：

- "把数字范围改成 1 到 1000"
- "背景改成深色，字改成白色"
- "猜对之后显示一个烟花效果"

每次 AI 给你新代码，全选复制，回到 JSFiddle 把之前的全删掉，粘贴新的，点 Run。

> **这就是 Vibe Coding 的核心循环**：你说想法 → AI 生成代码 → 你看到结果 → 你再提修改 → AI 再生成。

## 常见问题

**Q: AI 给我的代码点了 Run 没反应？**
A: 检查是否完整复制了所有代码（从 `<!DOCTYPE html>` 到最后）。如果还不行，按 F12 看 Console 里红色报错，把报错复制发给 AI："我运行你的代码后出现这个错误：[粘贴错误]，请修复。"

**Q: 游戏能玩，但是不好看？**
A: 告诉 AI："请把界面美化一下，加圆角、阴影、更好看的字体。"

**Q: 我想把这个游戏发给朋友玩怎么办？**
A: 把最新代码复制到记事本，保存为 `game.html`，双击用浏览器打开即可。把这个文件发给朋友，他也能玩。
