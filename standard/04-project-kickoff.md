# 第四章 新项目统一启动操作步骤

1. 新建项目根目录。
2. 部署硬约束文件（`CLAUDE.md`，Cursor 同时创建 `.cursorrules`，内容一致）。
3. 部署通用流程 Skill（`vibe-coding-std`），Cursor 放入 `.cursor/skills/`，VSCode 放入 `skills/`。
4. 从官网/GitHub 下载技术栈官方 Skill，放入 `skills/` 目录。
5. 初始化 `.gitignore` 与 Git 仓库，首次提交。
6. 严格按照 8 阶段流程从阶段 1 开始。
