# 阶段6：单模块全场景测试

## 概述

阶段6负责对每个模块进行全场景测试，确保模块具备完整容错能力。

## 核心任务

1. 覆盖正常流程、空数据、非法输入、超时、断连等异常场景
2. 前端组件必须实现loading/empty/error状态
3. 修复所有Bug

## AI引导话术

```
现在对模块进行全场景测试：

测试清单：
1. 正常流程测试
2. 空数据状态测试
3. 非法输入测试
4. 超时处理测试
5. 断连处理测试
6. Loading状态测试
7. Error状态测试
8. Empty状态测试

请逐项测试并告诉我结果。
```
