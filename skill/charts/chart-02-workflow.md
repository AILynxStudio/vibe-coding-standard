# 8 阶段开发流程图

生成一张横向流程图，1200×900px，白色背景，扁平化风格。

画布上方居中大号标题：Vibe Coding 8阶段工业级开发流水线

下方分为两行，每行 4 个圆角矩形框，箭头连接。

第一行（从左到右）：
- 阶段1：需求收敛。副标题：产出：简介+功能分级。
- 阶段2：架构锁档。顶部加红色五角星标注"核心锁档阶段"。副标题：产出：流程图+目录树+依赖清单。
- 阶段3：环境对齐。副标题：产出：校验通过说明。
- 阶段4：空项目初始化。顶部加红色五角星标注"硬性门槛"。副标题：产出：纯净骨架。下方画一条虚线箭头指向左侧，标注"启动失败则阻断"。

第二行（从左到右）：
- 阶段5：单模块增量编码。副标题：禁止：批量生成、多模块。
- 阶段6：单模块容错测试。副标题：覆盖：正常+异常+边界。
- 阶段7：全局联调重构优化。副标题：消除：冗余+重复+风格统一。
- 阶段8：打包交付文档归档。副标题：产出：制品+README+迭代计划。

箭头连接顺序：阶段1→2→3→4→5→6→7→8，均为实线箭头，灰色，2px。

画面最底部一行灰色文字：每个阶段结束 → 【阶段总结 + 下一阶段待办】→ 人工确认 → 下一阶段

所有阶段框使用浅蓝 #4299e1 描边，内部白色背景，黑色文字。
