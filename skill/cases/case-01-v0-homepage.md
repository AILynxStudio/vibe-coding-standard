# 案例1：v0 一句话生成个人主页，发布到互联网

> **难度**：★（零基础，只需浏览器）  
> **你需要**：一个浏览器，一个 v0.dev 账号（免费注册），一个 GitHub 账号（免费注册）  
> **你将做出**：一个属于自己的个人介绍网页，有一个网址可以发给任何人看

## 第 1 步：注册两个账号（5 分钟）

1. 打开 **v0.dev**，用 GitHub 账号直接登录。
2. 如果还没有 GitHub 账号，先去 **github.com** 注册一个。

## 第 2 步：用一句话生成你的主页

在 v0 的输入框里，输入你的需求。下面两个模板可复制，把括号内容改成你自己的：

**模板 A（极简风格）**：
> 创建一个个人介绍网页。名字是[你的名字]，标题是"[你的职业/兴趣]"。包含一个圆形头像占位图，一段自我介绍（大约50字），下方有3个按钮链接到我的 GitHub、微博和邮箱。风格极简，白色背景，深灰文字，蓝色强调色。

**模板 B（个性风格）**：
> 为我做一个个人主页。名字[你的名字]，我是[你的身份/爱好]。需要头像、关于我、我的技能（列出3-5项）、联系我。风格现代，渐变背景，卡片式布局。

输入后按回车，v0 会在几十秒内生成网页预览。

## 第 3 步：迭代修改，直到你满意

在右侧预览窗口下方的聊天框继续对话修改：
- "把头像换成一只猫的 emoji"
- "背景改成浅蓝色"
- "技能部分用进度条展示"
- "添加一个'我的项目'展示区，先放 3 个占位项目"

每次说完，v0 自动更新预览。

## 第 4 步：一键发布到互联网

点击 v0 界面右上角的 **Deploy** 按钮。v0 会自动生成一个网址（如 `https://你的项目名.vercel.app`），点开就能在任何设备上看到你的个人主页。把链接发给朋友，他们也能访问。

## 第 5 步（可选）：导出代码，留待以后继续开发

点击 v0 的 **Export Code** 按钮，选择导出为 React 或 HTML 代码包，下载到电脑上。以后在案例3中我们会用到。

> **小结**：你在没有写一行代码的情况下，完成了网页的设计、修改和发布。这就是 Vibe Coding 的威力。下一步，案例2会让你开始接触"用文件夹管理项目"。

## 常见问题

**Q: v0 生成的网页有乱码？**  
A: 跟 AI 说："我看到的网页上中文显示为乱码，请修复编码。"

**Q: 部署后网址打不开？**  
A: 刚部署完可能需等 1-2 分钟。如果长时间打不开，检查 v0 是否提示部署出错，把错误信息发给 v0 聊天框修复。

**Q: 我想绑定自己的域名？**  
A: 在 Vercel 控制台（vercel.com）找到项目，Settings → Domains 按提示添加。这是进阶操作，可以以后再做。
